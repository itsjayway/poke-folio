Introduction

Different programming languages can look quite different! Take a look at the following example:

public static void main(String[] args) { System.out.println("wow!"); }
**VERSUS** 
if __name__ == __main__:
   print("wow!")

Quite a difference, right? The second may look simpler, but everything has their pros and cons.

Declaring variables

In Java, you have to define the datatype of the variable you're creating. In python, you can just give it a name.

//Java
int my_number = 0;
String my_string = "hello";
boolean my_bool = true;

#Python
my_number = 0;
my_string = "hello"
my_bool = True;

Remember, indicate which datatype your variable will be and stick to it!
